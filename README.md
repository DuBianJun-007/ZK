# ZK algorithm  

  &emsp;  **A high-performance string retrieval algorithm**   
  &emsp; &emsp;  This project is an example of the ZK algorithm,Call some common string retrieval algorithms to compare the time consumption of different algorithms for different amounts of data.  
     
       
       
        
  ###  This item has three retrieval retrieves under different data amounts  
  1.Text strings are randomly generated  
    &emsp; &emsp; &emsp;   The length of the target string is determined by user input, and the length of the pattern string is also determined by user input.
  
  ---
  
  2.Text strings are generated from dictionary files  
    &emsp; &emsp; &emsp;   According to the length of the pattern string entered by the user, it is randomly intercepted to match the pattern string.    
    
  ---
  
  3.Text strings are generated by gene files  (size: 800 million)   
    &emsp; &emsp; &emsp; The pattern string is randomly intercepted by the length of the previous user input, and then the pattern string is matched.    
      
  ---
