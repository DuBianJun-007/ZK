# ZK arithmetic  

  &emsp; &emsp; **A high-performance string retrieval algorithm**   
  &emsp; &emsp;  This project is an example of the ZK algorithm,Call some common string retrieval algorithms to compare the time consumption of different algorithms for different amounts of data.  
  ###  This item has three retrieval retrieves under different data amounts  
  1.Pattern strings are randomly generated  
    &emsp; &emsp; &emsp;   Generate a random target string with the user input length, and then intercept the pattern string length according to the user input. To retrieve it.  
  
  ---
  
  2.Pattern strings are generated from dictionary files  
    &emsp; &emsp; &emsp;   According to the length of the pattern string entered by the user, it is randomly intercepted to match the pattern string.    
    
  ---
  
  3.Pattern strings are generated by gene files  (size: 800 million)   
    &emsp; &emsp; The pattern string is randomly intercepted by the length of the previous user input, and then the pattern string is matched.    
      
  ---
